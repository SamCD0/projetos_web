Tag de hiperlink

A tag a href define um hiperlink(conexão), que é utilizado para venclar uma página a outra.

O atributo mais importante dessa tag é o href que define a página de destino que será exibida após o clique do mouse no link.

Por padrão, os links aparecerão da segunte forma em todos os navegadores:

um link não visitado é sublinhado de azul

Um link visitado é sublinhado de roxo

um link ativo é sublinhado de vermelho

Exemplo

Link para o arquivo h1-h6

Clique aqui visitar a página sobre Marcadores H1-H6

clique aqui para visitar a página do SENAI Suzano

Para que o link abra a página em uma nova guia devemos acrescentar o atributo target = "_blank" aninhando com o a href

Clique aqui para abrir o link em uma nova janela do navegador

clique no botão para abrir a página
